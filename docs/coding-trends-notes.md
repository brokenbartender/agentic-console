# 2026 Agentic Coding Trends Report Notes

Source: 2026 Agentic Coding Trends Report.pdf

Key takeaways:
- Shift from assistant tools to collaborative agent teams.
- Long-running agents can build larger systems end-to-end.
- Human oversight scales with intelligent collaboration.
- Agents expand to new surfaces and users, beyond IDEs.

Applied to this app:
- Added team orchestration (Planner/Builder/Reviewer) via `team <task>`.
- Added job store and `/api/jobs` to support long-running tracking.
- Added autonomy levels for scalable human oversight.
