# Permissions

Permissions are enforced via allowlists and approval gates.

## Filesystem

- `AGENTIC_ALLOWED_PATHS` limits filesystem access.

## Network

- `AGENTIC_ALLOWED_DOMAINS` limits outbound domains.

## Shell

- `AGENTIC_ALLOWED_SHELL` restricts shell commands.

## Verification

- `AGENTIC_ALLOWED_VERIFY` restricts verify commands used for `test_passes`.

## MCP

- `AGENTIC_ALLOWED_MCP` restricts MCP provider names.
